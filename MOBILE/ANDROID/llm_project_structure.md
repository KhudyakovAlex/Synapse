# Структура проекта

## Package структура
Физические папки соответствуют package:
- `pages/` → `package com.awada.synapse.pages`
- `components/` → `package com.awada.synapse.components`
- `ai/` → `package com.awada.synapse.ai`
- `data/` → `package com.awada.synapse.data`
- `activities/` → `package com.awada.synapse.activities`
- `ui/theme/` → `package com.awada.synapse.ui.theme`

## Иконки (res/drawable/)

- `ic_ai.xml` — AI on
- `ic_ai_off.xml` — AI off
- `ic_back.xml` — кнопка назад
- `ic_backspace.xml` — удаление символа (клавиатура)
- `ic_microphone.xml` — микрофон on
- `ic_microphone_off.xml` — микрофон off (disabled)
- `ic_send.xml` — отправить сообщение
- `ic_settings.xml` — настройки
- `ic_volume.xml` — звук on
- `ic_volume_off.xml` — звук off
