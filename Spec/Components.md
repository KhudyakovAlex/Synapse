# Описание компонентов системы

[◀ Вернуться к диаграмме](Spec.md)

---

## MOBILE (Мобильное приложение)

### UI (User Interface)

Пользовательский интерфейс для взаимодействия с системой.

### CHAT

Чат-интерфейс для взаимодействия с LLM (языковой моделью).

### RM (Resource Manager)

Менеджер ресурсов - центральный компонент мобильного приложения, управляющий связями между компонентами.

### USMS (User Settings Management System)

Система управления пользовательскими настройками.

---

## CONTROLLER (Контроллер)

### RC (Resource Controller)

Контроллер ресурсов - центральный узел управления на стороне контроллера.

### AUTO

Модуль автоматизации.

### USMC (User Settings Management Controller)

Контроллер управления пользовательскими настройками.

### DALI (Digital Addressable Lighting Interface)

Интерфейс для управления цифровым адресуемым освещением.

---

## Внешние компоненты

### LLM (Large Language Model)

Большая языковая модель для обработки текстовых запросов.

### LINE

Линия связи / шина данных для управления устройствами.

---

## Связи между компонентами

Подробная схема связей доступна в [диаграмме системы](Spec.md).

**Основные взаимодействия:**

- LLM подключен к CHAT для обработки запросов
- RM является центральным узлом MOBILE
- RC является центральным узлом CONTROLLER
- DALI управляет LINE (устройствами)
