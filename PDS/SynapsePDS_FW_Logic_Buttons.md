# Прошивка. Логика работы. Управление настенными кнопками

АПК Синапс v1.0. ПО. Спецификации на разработку

**Последнее изменение:** 13.12.2025, 16:48 МСК

## 1. Короткое нажатие

1.1. При коротком нажатии кнопки прошивка получает событие от DALI.

1.2. Прошивка находит действие с **ACTIONS.BUTTON_SHORT_ID** = `i` и минимальным **POS**.

1.3. Прошивка выполняет найденное действие.

1.4. При следующем коротком нажатии прошивка находит действие с тем же **BUTTON_SHORT_ID** и следующим **POS** (по кругу).

## 2. Долгое нажатие

2.1. При долгом нажатии кнопки прошивка получает событие от DALI.

2.2. Прошивка находит действие с **ACTIONS.BUTTON_LONG_ID** = `i`.

2.3. Прошивка выполняет найденное действие.
