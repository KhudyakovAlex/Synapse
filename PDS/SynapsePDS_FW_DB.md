# Прошивка. База данных

АПК Синапс v1.0. ПО. Спецификации на разработку

**Последнее изменение:** 04.12.2025

## 1. Термины и определения

1.1. **Рабочие** данные — необходимые для организации работы освещения и в прошивке, и в приложении. В приложении находятся в таблицах SQLite, в прошивке в виде массивов структур C.

1.2. **Интерфейсные** данные — используемые только приложением для организации пользовательского интерфейса.  Примеры: названия светильников, номера их иконок.. В приложении находятся в таблицах SQLite, в прошивке в виде единого блока где-то в постоянной памяти.

## 2. Основные положения

2.1. «Таблицы» БД в прошивке разложены в массивы структур C. Тем не менее ниже для простоты они представляются именно как таблицы.

## 3. Таблицы

### 3.3. CONTROLLERS

Корневая таблица с информацией о контроллере. Имеет одну запись.

- **NAME** (char[20]) — название контроллера, показываемое в приложении и списке устройств Bluetooth;
- **PASSWORD** (char[4]) — пароль заводской или изменённый пользователем; 4 цифры; заводской на коробке (или наклейке на корпусе);
- **IS_SCHEDULE** (_Bool) — T/F — включена ли в контроллере работа по расписанию;
- **IS_AUTO** (_Bool) — T/F — главный выключатель автоматического режима для всего контроллера;
- **ICO_NUM** (uint8_t) — номер иконки, которую будет иметь контроллер в приложении; 0 — дефолтная.

### 3.4. LOCATIONS

Локации.

- **NUM** (uint8_t) — номер локации для идентификации локации в прошивке;
- **IS_AUTO** (_Bool) — работает ли локация в автоматическом режиме по датчикам (0/1).

### 3.5. LUMINAIRES

Светильники.

- **DALI_ADDR** (uint8_t) — короткий адрес DALI;
- **LOCATION_NUM** (uint8_t) — идентификатор локации для привязки светильника к ней в прошивке; 0 — светильник без локации (в корне);
- **GROUP_NUM** (uint8_t) — номер группы в DALI для привязки светильника в прошивке; если светильник не в группе — 255;
- **VAL_BRIGHT** (uint8_t) — значение яркости;
- **VAL_TW** (uint8_t) — значение температуры белого света;
- **VAL_R** (uint8_t) — значение красного канала (для RGB);
- **VAL_G** (uint8_t) — значение зелёного канала (для RGB);
- **VAL_B** (uint8_t) — значение синего канала (для RGB);
- **VAL_W** (uint8_t) — значение белого канала (для RGBW).

### 3.6. SCENE_LUMINAIRES

Параметры светильников в сценах.

- **DALI_ADDR** (uint8_t) — адрес светильника в DALI;
- **SCENE_NUM** (uint8_t*) — номер сцены в DALI;
- **VAL_BRIGHT** (uint8_t) — значение яркости;
- **VAL_TW** (uint8_t) — значение температуры белого света;
- **VAL_R** (uint8_t) — значение красного канала (для RGB);
- **VAL_G** (uint8_t) — значение зелёного канала (для RGB);
- **VAL_B** (uint8_t) — значение синего канала (для RGB);
- **VAL_W** (uint8_t) — значение белого канала (для RGBW).

### 3.7. GROUPS

Группы светильников.

- **LOCATION_NUM** (uint8_t) — идентификатор локации для привязки группы к ней в прошивке; 0 — группа без локации (в корне);
- **DALI_NUM** (uint8_t) — номер группы в DALI.

### 3.8. PRES_SENSORS

Датчики присутствия.

- **DALI_ADDR** (uint8_t) — короткий адрес DALI датчика;
- **DALI_INST** (uint8_t) — номер инстанса датчика движения в DALI;
- **LOCATION_NUM** (uint8_t) — идентификатор локации для привязки датчика к ней в прошивке; 0 — датчик без локации (в корне);
- **ACTION_OCCUPANCY_NUM** (uint8_t) — номер действия для прошивки; 0 — не повешено действие;
- **ACTION_VACANCY_NUM** (uint8_t) — номер действия для прошивки; 0 — не повешено действие;
- **DELAY** (uint8_t) — задержка HOLD_TIME.

### 3.9. BRIGHT_SENSORS

Датчики освещённости.

- **DALI_ADDR** (uint8_t) — короткий адрес датчика в DALI;
- **DALI_INST** (uint8_t) — номер инстанса датчика движения в DALI;
- **LOCATION_NUM** (uint8_t) — идентификатор локации для привязки датчика к ней в прошивке; 0 — датчик без локации (в корне);
- **GROUP_NUM** (uint8_t) — номер группы в DALI, которой управляет датчик; 255, если не назначена группа.

### 3.10. BUTTON_PANELS

Кнопочные панели.

- **DALI_ADDR** (uint8_t) — короткий адрес панели в DALI;
- **LOCATION_NUM** (uint8_t) — идентификатор локации для привязки панели к ней в прошивке; 0 — панель без локации (в корне).

### 3.11. BUTTONS

Кнопки.

- **DALI_ADDR** (uint8_t*) — короткий адрес DALI панели;
- **DALI_INST** (uint8_t) — номер инстанса кнопки в DALI;
- **ACTION_SET_SHORT_NUM** (uint8_t) — действие по короткому нажатию; 0 — не повешено действие;
- **ACTION_LONG_NUM** (uint8_t) — действие по долгому нажатию; 0 — не повешено действие.

### 3.12. ACTIONS

Действия с устройствами.

- **ACTION_SET_NUM** (uint8_t) — привязка к набору действий; если действие не входит ни в какой набор - 0 (для прошивки);
- **NUM** (uint8_t) — номер действия для идентификации действия в прошивке.
- **POS** (uint8_t) — номер по порядку действия (начиная с 1) в списке действий, привязанных к кнопке (случай перебора действий одной кнопкой)

### 3.13. SUBACTIONS

Поддействия с устройствами (команды для пользователя).

- **ACTION_NUM** (uint8_t) — действие (для прошивки);
- **OBJECT_TYPE** (uint8_t) — тип объекта изменений:
    - 1 — весь объект;
    - 2 — локация;
    - 3 — группа;
    - 4 — светильник;
- **OBJECT_NUM** (uint8_t) — номер объекта изменений:
    - 1 — сцена;
    - 2 — температура TW;
    - 3 — флаг АВТО.
- **VALUE** (uint8_t) — значение в зависимости от объекта:
    - сцена - 0..4;
    - температура TW - 0..254;
    - флаг АВТО - 0 / 1.


### 3.14. ACTION_SETS

Наборы действий для навешивания на короткие нажатия кнопок (когда перебираем действия одной кнопкой).

- **NUM** (uint8_t) — номер набора действий для идентификации в прошивке; начинается с 1.

### 3.15. EVENTS

События расписания.

- **DAYS** (char[7]) — 'FFTFFFF' — событие по дням недели; если все F — ежедневное;
- **TIME** (char[4]) — время события с точностью до минуты в формате HHMM 24H;
- **SMOOTH** (_Bool) — плавное изменение параметров от предыдущих значений (яркость, температура света);
- **ACTION_NUM** (uint8_t) — действие (для прошивки).
