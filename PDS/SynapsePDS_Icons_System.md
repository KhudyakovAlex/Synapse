## 1. Общие положения

1.1. Названия файлов с иконками имеют следующий формат: XXX_Nazvanie_ikonki.svg, где:  

- XXX — уникальный числовой идентификатор.  
- Nazvanie_ikonki — название иконки транслитерацией без пробелов.  

1.2. У системных иконок идентификатор начинается с 900 и заканчивается 999. 

1.3. Идентификатор 900 — дефолтная иконка.

1.4. По идентификатору иконка используется в интерфейсе приложения, поэтому на иконке с использовавшимся идентификатором содержимое должно сохраняться при обновлении версии приложения.

1.5. В новых версиях можно добавлять новые иконки, но только с новыми идентификаторами.
