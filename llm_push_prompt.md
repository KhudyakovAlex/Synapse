# Веб-версия репозитория

Кроме заливки локальной версии репозитория на GitHub нужно сделать (или обновить) веб-версию репозитория.

## Требования

### Структура

**Вариант 1: Папка INDEX/ (рекомендуемый)**
- Создать папку `INDEX/` в корне репозитория
- Все файлы веб-версии лежат в `INDEX/`
- Структура папок в `INDEX/` повторяет структуру репозитория
- Главная страница: `INDEX/index.html`
- GitHub Pages публикует из папки `INDEX/`

Структура:
```
D:\Git\Synapse\
├── index.html              (текущая страница-навигатор)
├── PRD/                    (исходные MD-файлы)
├── PDS/                    (исходные MD-файлы)
└── INDEX/                  (веб-версия)
    ├── index.html          (главная веб-версии)
    ├── PRD/
    │   ├── SynapsePRD.html
    │   └── Идеи.html
    ├── PDS/
    │   ├── SynapsePDS_APP.html
    │   └── ...
    └── assets/             (стили, скрипты)
        ├── css/
        ├── js/
        └── img/
```

**Вариант 2: Корень репозитория (альтернатива)**
- Веб-версия находится прямо в корне
- `index.html` в корне — главная страница веб-версии
- HTML-файлы рядом с MD-файлами
- GitHub Pages публикует из корня `/`

**Вариант 3: Папка docs/**
- Создать папку `docs/` в корне
- GitHub Pages часто использует эту папку по умолчанию
- Аналогично варианту с `INDEX/`

### Конвертация документов
- Все Markdown-файлы (*.md) должны иметь HTML-вариант
- Сохранять форматирование: заголовки, списки, таблицы, блоки кода
- Конвертировать Mermaid диаграммы в интерактивный вид
- Сохранять относительные ссылки между документами

### Навигация
- Главная страница с навигацией по всем разделам
- Хлебные крошки для навигации
- Боковое меню с оглавлением документа
- Кнопки "Назад к списку" / "На главную"
- Ссылки на исходные MD-файлы в репозитории

### Оформление
- Единый стиль для всех страниц
- Адаптивный дизайн (mobile-friendly)
- Подсветка синтаксиса кода
- Иконки для разных типов документов
- Тёмная и светлая темы

### GitHub Pages
- Настроить публикацию из папки `INDEX/` или через gh-pages branch
- URL: https://khudyakovalex.github.io/Synapse/

### Автоматизация
- Скрипт для автоматической конвертации MD → HTML
- При коммите в main/master — автоматическое обновление веб-версии

## Файлы для конвертации

### PRD (Product Requirements Document)
- `PRD/SynapsePRD.md` — основной документ с требованиями
- `PRD/Идеи.md` — дополнительные идеи

### PDS (Product Design Specification)
- `PDS/SynapsePDS_APP.md` — спецификация приложения
- `PDS/SynapsePDS_APP_UI.md` — UI приложения
- `PDS/SynapsePDS_DB.md` — база данных
- `PDS/SynapsePDS_FW.md` — прошивка
- `PDS/SynapsePDS_LLM.md` — интеграция с LLM
- `PDS/db_schema_mermaid.md` — схема БД (Mermaid)

### Корень
- `README.md` — описание проекта (если будет создан)
