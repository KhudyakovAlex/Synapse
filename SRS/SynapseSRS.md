# АПК Синапс. ПО. ТЗ

**Версия документа:** 1.0  
**Дата создания:** 11.11.2025  
**Статус:** Черновик

## 1. Назначение

1.1. Пуско-наладка, настройка и оперативное управление осветительным оборудованием по стандарту DALI.

## 2. Функции комплекса

2.1. Пуско-наладочные работы с оборудованием, подключаемым к линии DALI.

2.2. Настройка режимов работы оборудования под требования пользователя.

2.3. Оперативное управление оборудованием.

2.4. Автоматическая работа освещения по датчикам присутствия и освещенности.

2.5. Работа оборудования по расписанию.

## 3. Термины и определения

3.1. **АПК** - аппаратно-программный комплекс.

3.1. **Контроллер** - устройство на базе микроконтроллера GD32F103RBT6, работающий с устройствами DALI.

3.2. **Прошивка** - firmware в контроллере

3.3. **Телефон** - мобильное устройство, обеспечивающее пользовательский интерфейс АПК.

3.4. **Приложение** - интерфейсное мобильное приложение в телефоне.

3.5. **LLM** — Large Language Model (большая языковая модель).

3.6. **Локация** — помещение или часть помещения, с отдельно от других частей объекта организованной работой освещения.

## 4. Аппаратная архитектура АПК

4.1. В АПК входят контроллер и один, два или несколько телефонов.

4.2. Телефоны подключаются к контроллеру по Bluetooth.

4.3. С контроллером одновременно могут работать (быть подключены) несколько телефонов.

4.4. К контроллеру подключается 1 линия DALI, устройствами в которой управляет АПК.

## 5. Устройства

5.4. Количество устройств ограничено емкостью линии DALI:

- 64 светильника.
- 64 устройства управления (датчики, кнопки).

## 5. Локации

5.1. Устройства распределяются по локациям.

5.2. Устройство должно входить в локацию и только в одну.

5.3. Количество локаций не ограничено.

## 6. Группы

5.5. Светильники в рамках локации могут быть объединены в группы.

5.5. Группы в АПК соответствуют DALI-гпуппам.

5.6. Общее Максимальное количество групп на все локации - 16.

5.6. Светильник может входить только в одну группу или не входить ни в одну.

5.7. Светильники, объединенные в группу, получают возможность группового управления.

## 6. Световые сцены

6.1. Сцены в АПК соответствуют DALI-сценам.

5.6. 5 сцен (DALI-сцены 0..4) распространяются на все светильники в АПК.

6.3. По умолчанию сцены 0..4 - яркости всех светильников: 0, 25, 50, 75, 100%.

6.4. 5 сцен (DALI-сцены 5..9) распространяются на все светильники в группах.

6.3. По умолчанию сцены 5..9 - яркости всех светильников в группах: 0, 25, 50, 75, 100%.

6.6. С помощью приложения сценам 0..4 могут быть назначены произвольные яркости всех светильников АПК.

6.7. С помощью приложения сценам 5..9 для каждой из групп могут быть назначены произвольные яркости светильников, входящих в эти группы.

## 6. Функции прошивки

6.1. Полный сбор (пересбор) линии DALI со сбросом настроек устройств к заводским по команде от мобильного приложения.

6.2. Добавление новых устройств в линию без пересбора линии.

6.3. Замена вышедших из строя устройств без пересбора линии:

- подключение новых устройств к линии
- поиск устройств
- сопоставление их вышедшим из строя (с разрешением неоднозначностей пользователем)
- перенос старых настроек в новые устройства (название, короткий адрес DALI, FADETIME и т.д.)
