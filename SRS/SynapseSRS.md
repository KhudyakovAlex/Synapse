# АПК Синапс. ПО. ТЗ

**Версия документа:** 1.0  
**Дата создания:** 11.11.2025  
**Статус:** Черновик

## 1. Назначение

1.1. Пуско-наладка, настройка и оперативное управление осветительным оборудованием по стандарту DALI.

## 2. Функции комплекса

- Пуско-наладочные работы с оборудованием, подключаемым к линии DALI.
- Настройка режимов работы оборудования под требования пользователя.
- Оперативное управление оборудованием.
- Автоматическая работа освещения по датчикам присутствия и освещенности.
- Работа оборудования по расписанию.

## 2. Термины и определения

1.1. **Контроллер** - устройство на базе микроконтроллера GD32F103RBT6, работающий с устройствами DALI.

1.2. **Прошивка** - firmware в контроллере

1.3. **Телефон** - мобильное устройство, обеспечивающее пользовательский интерфейс АПК.

1.4. **Приложение** - интерфейсное мобильное приложение в телефоне.

1.5. **LLM** — Large Language Model (большая языковая модель).

1.6. **Локация** — помещение или часть помещения, с отдельно от других частей объекта организованной работой освещенгия.

## 3. Аппаратная архитектера АПК

3.1. В АПК входят контроллер и один, два или несклько телефонов.

3.2. Телефоны подключаются к контроллеру по Bluetooth.

3.3. С контроллером одновременно могут работать (быть подключены) несколько телефонов.

3.4. К контроллеру подключается 1 линия DALI, устройствами в которой управляет АПК.

## 4. Организационная структура устройств

4.1. Устройства распределяются по локациям.

4.2. Устройство должно входить в локацию и только в одну.

4.3. Количество локаций не ограничено.

4.4. Количество устройств ограничено емкостью линии DALI:

- 64 светильника
- 64 устройства управления (датчики, кнопки)

## 5. Функции прошивки

4.1.2. Полный сбор (пересбор) линии DALI со сбросом настроек устройств к заводским по команде от мобильного приложения.

4.1.3. Добавление новых устройств в линию без пересбора линии.

4.1.4. Замена вышедших из строя устройств без пересбора линии:

- подключение новых устройств к линии
- поиск устройств
- сопоставление их вышедшим из строя (с разрешением неоднозначностей пользователем)
- перенос старых настроек в новые устройства (название, короткий адрес DALI, FADETIME и т.д.)
