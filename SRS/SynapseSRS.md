# АПК Синапс. ПО. ТЗ

**Версия документа:** 1.0  
**Дата создания:** 09.11.2025  
**Статус:** Черновик

### 1. Введение

#### 1.1. Область применения
1.1.1. Пуско-наладка, настройка и оперативное управление осветительным и затеняющим оборудованием, поддерживающим стандарт DALI.

#### 1.2. Определения и сокращения
1.2.1. **LLM** — Large Language Model (большая языковая модель)

### 2. Общее описание

#### 2.1. Описание продукта
2.1.1. Синапс - аппаратно-программный комплекс, состоящий из:
- контроллера, взаимодействующего с инженерным оборудованием по протоколу DALI
- мобильных телефонов с установленным приложением-клиентом Синапс, подключающихся к контроллеру по Bluetooth и предоставляющих пользовательский интерфейс к комплексу.

#### 2.2. Функции продукта
- Пуско-наладочные работы с оборудованием, подключаемым к линии DALI.
- Настройка режимов работы оборудования под требования пользователя.
- Оперативное управление оборудованием.
- Автоматическая работа освещения по датчикам присутствия и освещенности.
- Работа оборудования по расписанию.

### 3. Функциональные требования к комплексу в целом

3.1.1. Поддержка до 15 локаций.

### 4. Функциональные требования к ПО контроллера

#### 4.1. DALI

4.1.1. Количество единиц оборудования - до 64 шт. DALI и 64 шт. DALI-2.

4.1.2. Полный сбор (пересбор) линии DALI со сбросом настроек устройств к заводским по команде от мобильного приложения.

4.1.3. Добавление новых устройств в линию без пересбора линии.

4.1.4. Замена вышедших из строя устройств без пересбора линии:
- подключение новых устройств к линии
- поиск устройств
- сопоставление их вышедшим из строя (с разрешением неоднозначностей пользователем)
- перенос старых настроек в новые устройства (название, короткий адрес DALI, FADETIME и т.д.)

4.1.5. Использование стандартных сцен и групп DALI для настройки сцен в мобильном приложении и разбивки устройств по локациям.

#### 4.2. Светильники

4.2.1. 1 группа и 5 световых сцен в каждой локации.

4.2.2. Возможность исключить светильник из группы и сцен локации и, как следствие, из автоматической работы.

#### 4.3. Кнопки

4.3.1. Возможность назначения действия кнопки на:
- включение одной из сцен в локации
- вкл/выкл произвольных светильников, исключенных из группы в локации
- включение одной из сцен для всего объекта
